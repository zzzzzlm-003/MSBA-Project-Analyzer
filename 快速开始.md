# 快速开始指南

## 三步开始使用

### 第一步：安装依赖

```bash
pip install -r requirements.txt
```

### 第二步：配置API密钥

创建 `.env` 文件：

```env
OPENAI_API_KEY=sk-your-api-key-here
```

### 第三步：运行分析

**方式1：使用配置文件（推荐）**

1. 编辑 `project_analyzer_config.json`，添加所有Google Drive链接
2. 运行：
```bash
python quick_start.py
```

**方式2：交互式运行**

```bash
python project_analyzer.py
```

然后选择：
- 选项1：手动输入链接
- 选项2：从配置文件读取
- 选项3：分析本地PDF文件

## 输出结果

分析完成后，结果保存在 `data/output/` 目录：
- Excel文件：`项目分析_YYYYMMDD_HHMMSS.xlsx`
- JSON文件：`项目分析_YYYYMMDD_HHMMSS.json`（用于Notion导入）

## 导入Notion（可选）

1. 在 https://www.notion.so/my-integrations 创建集成
2. 获取API密钥和数据库ID，添加到 `.env`：
```env
NOTION_API_KEY=secret_xxx
NOTION_DATABASE_ID=xxx
```
3. 运行：
```bash
python notion_integration.py
```

## 常见问题

**Q: Google Drive下载失败？**
A: 确保链接是公开共享的，或者使用"任何有链接的人都可以查看"权限

**Q: AI提取失败？**
A: 检查OpenAI API密钥是否正确，账户余额是否充足

**Q: PDF解析失败？**
A: 确保PDF文件没有加密，可以尝试手动下载PDF到本地，然后使用选项3分析
